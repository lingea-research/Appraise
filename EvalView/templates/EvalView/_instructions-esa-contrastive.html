• Highlighting errors:
    • To highlight, select the text that contains an error; this will mark it as a minor error.
    • To edit, click on the selected text to change the error to major or remove the highlight.
    • To mark missing text, highlight the [MISSING] tag if the translation omits important text from the source.
  • Choosing error severity:
    • Minor errors are imperfections or stylistic issues that do not impact the core message (e.g., awkward phrasing).
    • Major errors confuse the meaning, misrepresent the source, of violate the message (e.g., incorrect information, confusing wording).
  • Important rules:
    • Multiple errors: use separate highlights for each error.
    • Hallucinations: if the translation contains extra text that is not in the source (and is not a sensible addition), highlight it and mark it as a major error.
    • Wrong language: if a translation is in the wrong language, highlight the entire text, mark it as a major error, and assign the lowest score.
    • Consistency: as you work, pay attention to translation consistency (e.g., for technical terms) across the entire document.
